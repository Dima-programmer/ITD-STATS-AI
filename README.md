# ITD-STATS-AI
AI‑Enhanced Analytics for итд.com — полная статистика, умные топы, ИИ‑советы

## Инструкция по получению API-key:
1. Регистрируемся на сайте openrouter.ai
2. Переходим по ссылке https://openrouter.ai/keys и создаём новый api key.
3. Полученный api-key будет выглядеть примерно так: `sk-or-v1-24429adadada845395835fdsfs8885353953`
4. Вставляем этот api-key в настройках ITD-STATS
5. **ГОТОВО!**
